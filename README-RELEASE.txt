## 安装方式

* 将本压缩包解压到 `<坦克世界安装目录>\res_mos\<游戏当前版本号>`
  例如 C:\Games\World_of_Tanks\res_mods\0.9.5
* 解压过程中如果有询问是否要覆盖文件, 一定要选择"是"
* 安装前请自行做好备份工作
* 假如安装本插件而发生任何奇怪的事情, 才没有人会负责呢
* README.txt 文件并不是插件正常工作的必要条件.
  可以自行删除
